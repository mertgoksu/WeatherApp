# WeatherApp
A basic Kotlin app to practice retrieving data from the Internet with Retrofit/RxJava
<br>

## Features
- Temperature, humidity, wind speed, latitude, longitude display <br>
- Search by city names <br>
- Swipe to update the content <br>

## Technologies Used
- MVVM structure <br>
- Retrofit <br>
- RxJava <br>
- Activities <br>
- XML <br>
